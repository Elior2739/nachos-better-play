# Nachos better play

A extenstion that updates nachos-il to more styled and useable video player


<img width="600" height="400" alt="unnamed" src="https://github.com/user-attachments/assets/91f36f67-c315-42c7-8ae5-ff92b7054999" />

## What Changed

- Changed the video player to a new one ([Plyr](https://github.com/sampotts/plyr))
- Picture in picture mode
- More keyboard shortcuts (Left Arrow - Rewind, Right Arrow - Seek, Up Arrow - Volume Up, Down Arrow - Volume Down)
- Video player volume won't "jump" everytime you change to the next part or show
- The video starts right away and from where you stopped last time.
- More frequent requests to the server the updates where you stopped last time.
- Next part button that shows up 30 seconds before the show ends
